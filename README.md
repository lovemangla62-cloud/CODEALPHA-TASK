# CODEALPHA-TASK Hotel Booking System
Overview

The Hotel Booking System is a simple Java console-based application developed to manage hotel room bookings. It allows users to search available rooms, book rooms, view booking details, and cancel bookings. The project demonstrates the use of Object-Oriented Programming (OOP) concepts and Java collections.

Features
Search available rooms by category (Standard, Deluxe, Suite)
Book a room with customer details
Simulated payment confirmation
View all current bookings
Cancel an existing booking
Simple and user-friendly menu-driven interface
Technologies Used
Java
Object-Oriented Programming (OOP)
ArrayList
Scanner Class
